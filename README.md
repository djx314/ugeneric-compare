ugeneric-compare
===============

Compile time compare between ugeneric and each raw framework.  

## Slick
Run
```scala
sbt>aclean
```
Run
```scala
sbt>slickRaw
```
and write the await time to txt.  

Run
```scala
sbt>slickCompare
```
and write the await time to txt.

Then get the slick module test result:
```scala
sbt>aclean
sbt>slickRaw

sbt>slickCompare
```
